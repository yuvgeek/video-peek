Events. Create or join to learn.

Events page provide users to access the list of available events to join. If you want to share your knowledge, you can create a new event and add to it. The events can be filtered out using Skill Category.

Oh yeah, Skills. List of categories.

All the events are grouped by Skill. This helps the user to filter out the events they wanted to learn/contribute.

And lastly, this one. Meeting Screen.

Multiple users can join the any event. With WebRTC, we get video access based on user permission before proceeding it. We provide End Call feature to exit from the existing event.
